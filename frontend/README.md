# 层级代理团队 - 前端应用

这个目录包含了层级代理团队项目的前端应用。

## 文件说明

- `index.html`: 原始的HTML前端页面
- `vue-app.html`: 基于Vue 3的前端页面，提供更好的用户体验和JSON格式化显示
- `server.py`: 前端服务器，用于提供静态文件

## 使用方法

### 启动前端服务器

```bash
cd frontend
python server.py
```

服务器将在 http://localhost:9000 启动。

### 访问Vue应用

在浏览器中访问 http://localhost:9000/vue 即可使用Vue版本的前端应用。

## Vue应用特性

1. **多行文本输入区域**：可以输入复杂的多行查询需求
2. **递归限制设置**：可以调整递归限制参数
3. **JSON格式化显示**：自动检测并格式化JSON响应，使其更易于阅读
4. **流式响应**：实时显示后端API的流式响应
5. **响应清空功能**：可以清空当前的响应内容
6. **加载状态指示**：在请求处理过程中显示加载指示器

## 注意事项

- 使用Vue应用前，请确保后端API服务器已经启动（默认地址为 http://localhost:8000）
- 如果需要修改API地址，请编辑 `vue-app.html` 文件中的 `API_BASE_URL` 变量
- 响应超时时间设置为5分钟，如需调整，请修改 `vue-app.html` 文件中的超时设置
